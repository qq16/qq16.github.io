## 自我介绍

博主 2020 年 6 月毕业，之前与之后参与的项目用到的前端框架都是 react，
现在因为一些原因开始学习 vue。

## 参与过的几个项目

1. 网页设计与制作课的教育教学相关主题网站(期末考查作业)
2. 一个`ionic`商城 App(设计目标不止是商城，但是优先实现方便实现的)
3. 一个信息管理系统网站附带的 App(使用`React Native`)
4. 一个链上数据分析网站(用了`recoil`和`tailwindCss`，后期改用 vue 技术栈)

## 谈谈收获

期末考查作业是列出的 4 个项目中唯一一个在毕业前做的项目。在这个作业之前，
我也做过一些简单的`SPA`，不过只是了解了一些知识，却没有深度运用。
通过这个项目，我接触到了一些新的东西，比如`ant design`,`normalize.css`等。
对之前接触过的轮子比如`react router`也有了更深的了解。（一点后话：当时路由传参得用`props.match.params`，现在可以用`useParams`这个钩子函数，方便多了)

`ionic`商城App是2020年7月开始做的，因为之前较少接触移动开发
而且完全没用过`ionic`框架，于是只能一边学一边开发。
`ionic` 一开始只限于用`angular`开发，后面用任何前端技术都行，
还出现了ionic-react、ionic-vue这样的组件库。虽然这个项目的节奏很紧，
压力很大，但是我在编码速度和心理素质等各方面都得到了锻炼。

因为2018年我就有学习过一点`react native`的基本用法，
而且RN的资料和轮子生态比ionic丰富多了,
所以之后用`react native`开发一个工业园助手App也就不是特别吃力。

之后的链上数据分析网站后期改用了vue技术栈，我也发现了react主要是外企喜欢用，
找工作还是学vue比较实在，
于是回到了之前上学的地方（长沙市岳麓区）参与到了一个使用vue技术栈的项目，
是一个关于房源的信息管理系统。
在这个项目中我对element ui和vuex的了解增多了不少。
